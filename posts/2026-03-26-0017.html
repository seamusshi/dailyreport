Cisco 推出 DefenseClaw，为企业 OpenClaw 部署保驾护航

2026-03-24

Cisco 正式发布 DefenseClaw，这是一款专门针对 OpenClaw 企业部署的安全解决方案。该产品提供三层防护机制：准入扫描（安装前检测恶意技能）、运行时检测（监控消息流）、强制黑白名单（2秒内拦截威胁）。DefenseClaw 与 Splunk 无缝集成，实现全程可观测，将于3月27日在 GitHub 开源发布。

OpenClaw 安全漏洞研究引发关注

2026-03-16

ArXiv 发表研究论文《Trojan's Whisper》，披露了针对 OpenClaw v2026.3.12 的安全评估。研究团队设计了26个对抗性技能，测试了包括 Claude Opus 4.6、GPT-5.3-Codex、Kimi K2.5 在内的6个大模型后端，发现 Agent 在执行复杂任务时存在潜在的安全风险。研究提醒企业用户在生产环境部署时应加强权限控制和输入验证。
