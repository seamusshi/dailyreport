🔷 OpenClaw 官方动态

OpenClaw v2026.3.12 正式发布

3月13日，OpenClaw 发布 v2026.3.12 版本，带来多项重磅更新：

  • Control UI/dashboard-v2：全新网关仪表盘上线，支持模块化概览视图、聊天、配置、Agent 和会话管理，新增命令面板、移动端底部导航、斜杠命令、搜索、导出和置顶消息功能
  • Fast Mode 支持：新增 OpenAI/GPT-5.4 Fast Mode 与 Anthropic Claude Fast Mode 配置，可通过 /fast 命令或 TUI、Control UI 快速切换
  • Provider 插件化：Ollama、vLLM、SGLang 迁移至 Provider 插件架构，支持提供商自有的 onboarding、模型发现和选择流程
  • sessions_yield：新增 sessions_yield 功能，允许编排器立即结束当前回合，跳过排队的工具工作，并将隐藏跟进载荷带入下一个会话回合
  • Kubernetes 部署：新增 Kubernetes 入门部署文档，包含原始清单、Kind 设置指南

OpenClaw v2026.3.8 安全更新

此前发布的 v2026.3.8 版本专注于安全与企业级功能：

  • ACP Provenance：新增代理通信协议溯源功能，可追踪每条指令的来源，防止身份伪造
  • 备份命令：内置 openclaw backup 功能，支持一键创建系统状态归档并验证可恢复性
  • Telegram 修复：消除重复事件，稳定聊天集成，减少冗余触发
  • VirusTotal 集成：Skills 安装前可扫描已知威胁，增强安全性
  • 12+ 安全补丁：强化网关重启恢复、配置验证等安全机制
